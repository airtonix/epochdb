How to contribute

Mostly you'll need a [github account](https://github.com/join), which is free.

Pull Requests

I'll be placing priority on these types of contributions

  1. Clone the repo
  2. Make changes, Limit changes to one feature/fix per pull request or it will be rejected.
  3. Submit a (pull request)[http://www.youtube.com/watch?v=YTbRzhQju4c]

Issues
  1. Head over to the issue tracker
  2. Write a brief description
  3. explain concisely the contribution
  4. submit it!
